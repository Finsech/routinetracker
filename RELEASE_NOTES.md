# Release Notes

## FocusFlow 0.1.2

Дата релиза: 2026-06-30

### Что вошло в обновление

- LLM-саммаризация длинных дней переведена на двухэтапную схему `chunk + merge`:
  - день больше не уходит в один слишком большой prompt;
  - локальная модель сначала собирает группы по чанкам;
  - затем отдельный merge-pass сводит их в итоговые потоки и стримы;
  - если merge-pass срывается, приложение показывает fallback из объединенных chunk-групп, а не сбрасывает результат целиком.
- Исправлена проблема с битой кодировкой в блоках LLM-сводки и связанных подписях интерфейса.
- Для потока `Общение` добавлена нормализация имен стримов до уровня мессенджера:
  - `Коммуникация в Telegram`
  - `Коммуникация в Slack`
  - `Коммуникация в Discord`
  - `Коммуникация в WhatsApp`
  - `Коммуникация в MAX`
  - `Коммуникация в Яндекс Мессенджере`
  - или `Коммуникация в чатах`, если источники смешаны.

### Что проверено

- production frontend build: `npm run build`
- Rust-проверка: `cargo check`
- Windows desktop bundle build: `npm run tauri build`
