# Orchestrator

Эта папка хранит отложенную спецификацию будущего оркестратора разработки для FocusFlow.

Сейчас основной проект разрабатывается **без оркестратора**. Решение принято осознанно: FocusFlow сам по себе достаточно сложный продукт, а полноценный LLM-оркестратор добавил бы второй большой проект до появления рабочего MVP.

К оркестратору стоит вернуться позже, когда появятся:

- стабильная структура проекта;
- повторяемый build/test pipeline;
- набор типовых задач;
- понятные правила безопасной автоматизации;
- реальная боль от ручных повторяющихся операций.

Основная спецификация продукта находится в `../SPECIFICATION.md` и не должна смешиваться со спецификацией оркестратора.
